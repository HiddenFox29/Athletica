# Сценарии использования (Use Cases — сценарии использования)

Документ содержит краткий обзор ключевых сценариев использования платформы Athletica.

Подробное описание каждого сценария, включая последовательности взаимодействия и архитектурные детали, вынесено в отдельные файлы.

---

## Use Case 1 — Регистрация и вход пользователя

Пользователь создаёт учётную запись в системе и выполняет вход.

Сценарий включает работу Auth Domain и Profile Domain и является точкой входа пользователя в систему.

---

## Use Case 2 — Запись тренировки

Пользователь или устройство передаёт данные тренировки, которые сохраняются в системе.

Сценарий охватывает Training Domain, Device Integration Domain и аналитический контур.

---

## Use Case 3 — Социальная активность

Пользователь публикует результат тренировки и взаимодействует с другими пользователями.

Сценарий реализуется через Social Domain и Notification Domain.

---

## Use Case 4 — Генерация рекомендаций

Система формирует персональные рекомендации на основе активности пользователя.

Сценарий включает Recommendation Domain и Analytics Domain.

---

## Use Case 5 — Интеграция с устройствами

Система принимает и обрабатывает данные от спортивных устройств.

Сценарий реализуется через Device Integration Domain и последующую обработку в Training и Analytics доменах.

---

## Вывод

Сценарии использования отражают ключевые бизнес-процессы платформы Athletica и служат отправной точкой для более детального описания взаимодействий в отдельных документах.
